4
Coller dans Claude avec une demande précise
La formule qui donne les meilleurs résultats

Dans une nouvelle conversation Claude, colle le code et décris la modification :

prompt type à copier
Voici un élément de ma page HTML : [colle ici le code copié depuis les DevTools] Modifie UNIQUEMENT cet élément pour que : → [décris ta modification précise] Retourne-moi juste le code HTML modifié, sans rien d'autre.

Exemple concret :

C
Claude · claude.ai
Voici un élément de ma page HTML :

<button class="btn-contact">Me contacter</button>

Modifie UNIQUEMENT cet élément pour que :
→ bouton arrondi (border-radius 50px), fond dégradé violet→bleu, texte en majuscules, ombre portée

Retourne juste le HTML modifié.
Voici ton bouton modifié :
<button class="btn-contact" style="
  background: linear-gradient(135deg, #7c3aed, #2563eb);
  border-radius: 50px;
  color: white;
  border: none;
  padding: 10px 22px;
  font-weight: 600;
  text-transform: uppercase;
  letter-spacing: 0.5px;
  box-shadow: 0 4px 14px rgba(124,58,237,0.35);
  cursor: pointer;
">ME CONTACTER</button>
Remplace l'ancien bouton par ce code dans ton fichier .html.
⚠️
Sois précis dans ta demande. Évite "rends-le plus joli" — préfère "change la couleur en #e94560, taille 16px, et ajoute une animation hover qui assombrit".
5
Remplacer dans le fichier et sauvegarder
Chercher l'ancien → sélectionner → coller → sauver → recharger

Ouvre ton .html dans ton éditeur (VS Code, Notepad++…) :

1
Ouvre le fichier → clic droit sur le .html → "Ouvrir avec…"
2
Recherche un texte unique : Ctrl+F → tape btn-contact
Recherche un texte unique : ⌘+F → tape btn-contact
3
Sélectionne tout l'ancien élément de la balise ouvrante <button jusqu'à </button>
4
Colle Ctrl+V → Sauvegarde Ctrl+S
Colle ⌘+V → Sauvegarde ⌘+S

Recharge la page pour voir le résultat :

F5ouCtrl+R
⌘+R
🚀
Dans VS Code, Ctrl+H ouvre le chercher-remplacer. Tu peux coller l'ancien HTML dans le champ "chercher" et le nouveau dans "remplacer" — une seule action.
