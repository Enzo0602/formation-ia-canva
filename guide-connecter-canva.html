Guide 5

Connecter Canva à Claude

Active le connecteur Canva dans Claude pour créer des designs, remplir des modèles et exporter directement depuis le chat.

1 Vérifier l'éligibilité › 2 Connecter le compte › 3 Activer dans le chat › 4 Faire la demande
1
Vérifier que ton compte Canva est éligible
Le connecteur IA nécessite un abonnement Canva Pro, Équipe, Pro+ ou Solidaire

Le connecteur Canva n'est pas disponible avec un compte Canva gratuit. Il fonctionne avec :

✅ Comptes éligibles
  • Canva Pro
  • Canva en Équipe (Teams)
  • Canva Pro+
  • Canva Solidaire (Nonprofits)
❌ Non éligible
  • Canva gratuit
  • Canva Éducation (accès limité)
🏢
Si tu fais partie d'une équipe Canva (Teams/Entreprise/Éducation), un admin doit aussi activer les "intégrations tierces" dans les contrôles d'administration pour que le connecteur fonctionne.
2
Connecter ton compte Canva à Claude
Paramètres → Connecteurs → Canva

Sur claude.ai, ouvre Paramètres puis l'onglet Connecteurs. Cherche Canva dans la liste et sélectionne-le.

3
Activer Canva dans une conversation
Icône des paramètres dans le chat → activer le connecteur Canva

Dans une nouvelle discussion, clique sur l'icône ⚙ Paramètres (ou l'icône d'outils) en bas de la zone de saisie. Dans la section Connecteurs, active le toggle Canva.

⚙ Outils disponibles dans cette discussion

🔄
Cette activation est par conversation. Si tu ouvres une nouvelle discussion et veux utiliser Canva, réactive simplement le toggle.
4
Faire ta demande à Claude
Crée, retrouve ou exporte un design directement depuis le chat

Une fois Canva activé, décris simplement ce que tu veux dans le chat :

💬 Exemples de demandes
Crée une présentation Canva de 5 slides pour présenter mon projet "Formation IA Canva", avec mon Brand Kit.
Retrouve mon design "Logo Hopsite" dans Canva et exporte-le en PNG.
Remplis automatiquement mon modèle "Flyer Atelier" avec ces informations : [...]

Claude crée ou modifie le design directement dans ton compte Canva — il te donne ensuite un lien pour l'ouvrir et le finaliser dans l'éditeur Canva.

✅ Ce que Claude peut faire
  • Créer des designs éditables (présentations, posts, flyers...)
  • Utiliser ton Brand Kit et tes modèles
  • Remplir automatiquement des modèles (autofill)
  • Retrouver des designs existants
  • Exporter en PDF ou image
❌ Ce que Claude ne fait pas
  • Générer des images par IA dans le design (utilise Canva AI pour ça)
  • Fonctionne sans abonnement Canva éligible
  • Autofill si l'option n'est pas incluse dans ton forfait
🔌
Pour déconnecter Canva : Paramètres → Connecteurs → Canva → ⋯ → Dissocier. Tu peux aussi le faire depuis Canva : Paramètres → Ton profil → Intégrations connectées → Déconnecter le connecteur IA.
