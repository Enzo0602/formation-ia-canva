Le nommage des fichiers
Une convention simple, à appliquer systématiquement

Un bon nom de fichier permet de retrouver n'importe quel document en quelques secondes, sans avoir à l'ouvrir.

[CSE]_[TYPE]_[SUJET]_[DATE]_[v1]
CSE_AFFICHE_Noel_2024-11_v1 CSE_NEWSLETTER_Fev2025_FINAL CSE_PRES_BilanAnnuel_2024_PRINT TEMPLATE_AFFICHE_A4_ORIGINAL

Gestion des versions — un suffixe qui indique toujours où en est le document :

v1

Première version de travail

v2

Après corrections du groupe

FINAL

Version validée par le bureau

PRINT

Version téléchargée pour impression

⚠️
Ne jamais supprimer les anciennes versions — archivez-les plutôt dans un dossier dédié par année.
4
Choisir le bon format d'export
PDF, JPG/PNG ou lien de partage — selon l'usage

📄 PDF pour l'impression

Toujours utiliser le PDF pour imprimer. PDF Standard pour un usage courant, PDF Premium pour une qualité impression professionnelle (fonds perdus, crops inclus).

🖼️ JPG ou PNG pour le numérique

JPG pour les photos et visuels envoyés par mail ou WhatsApp (fichier léger). PNG pour les images avec fond transparent (logo sur fond coloré). Parfait pour écran, réseaux sociaux et mails.

🔗 Lien de partage direct

Au lieu de télécharger, partagez un lien : le destinataire voit le design dans son navigateur. Pratique pour présenter en réunion ou faire valider rapidement — le lien reflète toujours la dernière version.

💡
Règle simple : Impression → toujours PDF. Mail ou réseaux → JPG ou PNG. Validation rapide → lien Canva. Téléchargez le PDF définitif après validation complète et relecture — jamais avant.
