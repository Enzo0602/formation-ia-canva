Guide 3

Publier un fichier HTML
sur GitHub Pages

Créer dans Claude, télécharger, uploader sur GitHub et obtenir son lien public — en 4 étapes.

1 Créer dans Claude
›
2 Télécharger
›
3 Uploader sur GitHub
›
4 Copier l'URL
1
Créer ton fichier HTML dans Claude
Demande à Claude de générer le contenu de ta page

Dans une conversation Claude, décris simplement ce que tu veux. Par exemple :

💬
« Crée-moi un fichier HTML pour présenter mon CV / mon projet / ma liste de liens… »

Claude génère le fichier et le rend téléchargeable directement dans le chat.

2
Télécharger le fichier
Note le nom du fichier — tu en auras besoin à l'étape 4

Clique sur le bouton de téléchargement sous le fichier généré par Claude. Retiens bien son nom (ex : mon-cv.html).

💡
Si ton fichier s'appelle index.html, l'URL sera plus simple — pas besoin d'ajouter le nom à la fin.
3
Uploader sur GitHub
Add file → Upload → Commit changes

Va dans ton dépôt GitHub. Clique sur Add file puis Upload files.

Laisse le message de commit par défaut ("Add files via upload") et clique Commit changes. C'est tout pour cette étape !
4
Récupérer l'URL publique
Settings → Pages → copier l'URL + ajouter le nom du fichier

Dans ton dépôt, va dans Settings (onglet en haut) puis dans la section Pages dans la barre latérale gauche.

GitHub affiche ton URL de base en vert. Il suffit maintenant d'ajouter le nom de ton fichier à la fin :

Ton URL finale
https:// ton-pseudo .github.io / nom-du-repo / mon-fichier.html

Remplace mon-fichier.html par le vrai nom de ton fichier tel qu'il apparaît dans la liste de ton dépôt.

⏱
Le déploiement prend 1 à 2 minutes après le commit. Si la page n'apparaît pas tout de suite, attends un peu et recharge.
