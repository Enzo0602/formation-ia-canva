Les 3 erreurs avec l'IA à ne plus jamais faire
À partir d'aujourd'hui

Erreur 1 — Copier sans relire

❌ « L'IA l'a écrit donc c'est forcément bon »
✅ Relisez systématiquement comme si c'était un brouillon de stagiaire — avec bienveillance mais avec rigueur.

Erreur 2 — Prompts trop vagues

❌ « Aide-moi pour mon affiche »
✅ Méthode PROMPT : Rôle + Tâche + Contexte + Format → résultat prêt à l'emploi.

Erreur 3 — Croire tous les chiffres

❌ « L'IA dit que le SMIC est X — c'est forcément à jour »
✅ Tout chiffre important → vérification sur legifrance.fr ou service-public.fr.
🤝
L'IA accélère votre travail. Votre jugement le valide. Ce duo — IA + humain — est toujours meilleur que l'un ou l'autre seul.

Votre boîte à outils complète
Les 4 méthodes essentielles de la formation — condensées en une page

Retrouvez ici, sur une seule page, les réflexes clés à garder sous la main après la formation.

🧩 Méthode PROMPT (Guide 1.6)

Rôle — « Tu es secrétaire du CSE / juriste… »
Tâche — « Rédige / Résume / Compare… »
Contexte — qui, quand, où, pour qui, enjeu
Format — longueur, ton, structure souhaitée

🔒 Réflexe 5 s avant l'IA (Guide 1.4)

Personne identifiable ? → rôle générique
Information confidentielle ? → reformuler ou ne pas envoyer
Outil ? → compte perso pour usages syndicaux
Chiffre important ? → vérifier sur une source officielle

✉️ Structure du mail parfait (Guide 2.5)

Objet — date + événement + bénéfice
Introduction — 1-2 phrases, bénéfice en premier
Corps — gras sur les éléments clés, liste si multi-info
CTA — une seule action demandée, clairement formulée

📁 Nommage Canva (Guide 3.2)

Convention : CSE_TYPE_SUJET_DATE_v1
Versions : v1 → v2 → FINAL → PRINT
Templates : dossier protégé, jamais modifié directement
Exports : PDF pour impression, JPG/PNG pour numérique
